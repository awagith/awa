Instruções Manuais (DevTools)

Chrome/Edge:
1. Abra DevTools (F12)
2. Vá em Application > Service Workers
3. Clique em "Unregister" em cada worker listado
4. Em Storage, clique em "Clear site data"
Firefox:
1. Digite about:debugging na barra de endereços
2. Clique em "This Firefox"
3. Encontre Service Workers e clique em "Unregister"
4. DevTools (F12) > Storage > Clear All

Console Alternativo

Cole este código no Console do DevTools (F12):

navigator.serviceWorker.getRegistrations().then(registrations => {
    registrations.forEach(registration => {
        console.log('Removendo SW:', registration.scope);
        registration.unregister();
    });
    console.log('Service Workers removidos!');
});

caches.keys().then(cacheNames => {
    cacheNames.forEach(cacheName => {
        console.log('Removendo cache:', cacheName);
        caches.delete(cacheName);
    });
    console.log('Caches limpos!');
});
