Pontos de Integração:

  ✅ Auto-Link na Aprovação
     → Observer: ErpApprovalSyncObserver
     → Evento: customer_approval (custom)
     → Fluxo:
       1. Admin aprova cliente B2B
       2. Observer dispara
       3. Busca CNPJ no ERP (CustomerSync::getErpCustomerByTaxvat)
       4. Se encontrado → faz link automático (erp_code)
       5. Sincroniza limite de crédito
       6. Importa endereços do ERP
       7. Define transportadora preferencial
       8. Move para grupo B2B apropriado

  ✅ Preços Específicos B2B
     → Plugin: GroupPricePlugin
     → Busca FATORPRECO do cliente no ERP
     → Aplica preço da tabela específica
     → Soma com desconto do grupo B2B
     → Preço final = Preço Tabela - Desconto Grupo

  ✅ Ocultar Preços sem ERP
     → Verifica se cliente tem erp_code
     → Se NULL → oculta preços
     → Mensagem: "Sua tabela de preços está sendo definida..."
     → Config: grupoawamotos_b2b/price_visibility/hide_price_no_erp

  ✅ Pedidos → ERP
     → Observer: OrderPlaceAfter
     → Queue: erp.order.sync
     → Consumer: erp.order.sync.consumer
     → Vincula erp_code do cliente
     → Insere em FN_PEDIDOS e FN_PEDIDOSITENS
     → Log completo de sincronização

  ✅ Limite de Crédito
     → Sincronizado do ERP na aprovação
     → Atualizado em customer attribute
     → Usado em CreditService

Atributos Criados:
  - erp_code (customer) → Código do cliente no ERP (CODIGO)
  - credit_limit (customer) → Limite de crédito do ERP
  - erp_sync_status (customer) → Status da sincronização
